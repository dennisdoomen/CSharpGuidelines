---
rule_id: 1710
rule_category: naming-conventions
title: Don't repeat the name of a class or enumeration in its members
severity: 1
---
```csharp
class Employee
{
    // Wrong!
    static GetEmployee() {...}
    DeleteEmployee() {...}

    // Right
    static Get() {...}
    Delete() {...}

    // Also correct.
    AddNewJob() {...}
    RegisterForMeeting() {...}
}
```
